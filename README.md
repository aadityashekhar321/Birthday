## 🎬 The Experience

The site is designed to feel like a vintage film premiere. From the moment it loads, you are greeted with a countdown that gives way to a cinematic curtain reveal, leading into a world of interactive storytelling.

### ✨ Key Features at a Glance

| Feature | Description | Interaction |
| :--- | :--- | :--- |
| **Cinematic Intro** | Countdown with a 1.8s curtain opening animation. | Auto-start on load |
| **Interactive Stars** | A parallax starfield that follows your mouse movement. | Mouse Hover |
| **Particle Bursts** | High-performance Canvas-based fireworks and sparkles. | Click Anywhere |
| **Memory Reel** | A fluid, Polaroid-style photo gallery with smooth rotations. | Navigation Buttons |
| **Roast Archive** | A hidden "Top Secret" vault containing playful roasts. | Toggle Button |
| **Animated Cake** | An SVG-drawn cake with flickering CSS flames. | Scroll Reveal |

## 🛠️ Technical Deep Dive

This project pushes the boundaries of standalone HTML files, packing complex logic and high-end styling into a single portable document.

### 🎨 Design System
- **Vintage Grained Overlay**: A fixed pseudo-element utilizing an animated film grain texture to give every section a consistent, "analog" feel.
- **Glassmorphism UI**: Extensive use of `backdrop-filter: blur()` and semi-transparent borders to create a modern, premium look.
- **Dynamic Color Palette**: Fuses **Amber**, **Rose**, and **Gold** gradients for a warm, celebratory atmosphere.

### 🧠 Core Mechanics
- **Canvas Particle System**: A custom-built JavaScript engine that manages dozens of `Particle` objects in real-time. It handles friction, gravity, and opacity decay to simulate realistic firework bursts.
- **Intersection Observer API**: All sections use a staggered "reveal" animation that triggers only when the user scrolls into view, ensuring a smooth performance even on lower-end devices.
- **State Management**: Handles complex UI states like the Roast Archive expansion and Polaroid card cycling through Vanilla JS.

## 🚀 Getting Started

Since this is a self-contained masterpiece, deployment is as simple as it gets.

1.  **Clone/Download**: Ensure you have `code.html` in your local directory.
2.  **Open**: Double-click the file to open it in Chrome, Firefox, or Safari.
3.  **Requirements**: An active internet connection is recommended to load:
    *   **Tailwind CSS** (via CDN)
    *   **Google Fonts** (Josefin Sans, Cormorant Garamond, Architects Daughter)
    *   **External Assets** (Memory reel images)
